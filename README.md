# AutoNova
## Introducció
Aquest projecte consta d'una web de compra i venta de cotxes de segona mà. En el front-end es farà ús de React i Tailwind,
mentres que en el back-end es farà ús de MongoDB i NodeJS.

Per a començar aquest projecte, el primer que he fet ha sigut inicialitzar el repositori de git i afegir aquest fitxer,
després he creat una carpeta per a fer tota la part de client per a començar a fer el front-end.
